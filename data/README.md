# Local dictionary data

`ecdict.sqlite3` is a compact subset of [ECDICT](https://github.com/skywind3000/ECDICT) for offline word and phrase lookup.

- Source: ECDICT
- License: MIT
- Selection: common corpus entries and short general-language phrases
- Rebuild: `python scripts/build_local_dictionary.py path/to/ecdict.csv`

The original ECDICT CSV is not required when running VReply.
